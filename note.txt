เราได้ทำการเขียนcodeขึ้นมาใน program ที่ชื่อว่า Visual Studio Code
ซึ่งคุณสามารถrun code ได้ตามหัวข้อดังกล่าว
1) index.html หน้าแรกของเว็บเพจ
2) street.html หน้าเกี่ยวกับสไตล์สตรีท
3) vintage.html หน้าเกี่ยวกับสไตล์วินเทจ
4) bohemain.html หน้าเกี่ยวกับสไตล์โบฮีเมียน
5) summer.html หน้าเกี่ยวกับสไตล์ซัมเมอร์
ถ้าเริ่ม run code อื่นนอกจากนี้ จะไม่ขึ้นตามที่เรากำหนดมา เพราะส่วนใหญ่จะเป็นส่วนประกอบ
ของหน้าที่เราได้กล่าวมาข้างต้น
